# House Price Statistical Validation & Data Storytelling Project

## Introduction

This project was completed as part of the ApexPlanet Data Analytics Internship Program.

The objective of Task 4 was to validate business insights through statistical analysis, hypothesis testing, and strategic data storytelling to transform housing market data into evidence-based business recommendations.

---

## Project Goals

* Perform statistical analysis on housing data
* Validate trends through hypothesis testing
* Identify significant business factors
* Analyze correlations
* Build data-driven recommendations
* Communicate insights through storytelling

---

## Technologies Used

* Python
* Pandas
* SciPy
* Statistical Analysis

---

## Project Structure

* `statistical_analysis.py` → Descriptive statistics and correlation analysis
* `hypothesis_testing.py` → T-test validation
* `data_storytelling_report.md` → Strategic business storytelling
* `cleaned_housing_data.csv` → Cleaned housing dataset

---

## Key Analysis Performed

### Statistical Analysis

* Descriptive summaries
* Correlation matrix
* Price segmentation
* Furnishing trend analysis

### Hypothesis Testing

* T-test between furnished and unfurnished house prices
* Statistical significance validation
* Market assumption testing

### Data Storytelling

* Strategic findings
* Business recommendations
* Market opportunity insights
* Executive summary reporting
